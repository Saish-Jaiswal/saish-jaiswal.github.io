# Assets

This folder contains static assets referenced by the website.

## Files

- **`profile.JPG`** — Profile photo (portrait image, ideally square, ~200×200px or larger)
  - Currently displayed in the About section
  - If missing, initials (SJ) are shown instead
  - Supported formats: JPG, PNG, WebP

- **`cv.pdf`** — Curriculum Vitae / Resume (PDF)
  - Linked from the About section and Contact card
  - Should be a single PDF file with your academic CV

## How to Add Files

1. Save your **profile photo** as `profile.JPG` in this folder
2. Save your **CV** as `cv.pdf` in this folder
3. Commit and push to GitHub — the site will automatically serve them

No additional configuration needed. The image and PDF paths are relative and will work on GitHub Pages.
